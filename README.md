# Sales Data Analysis & Customer Purchasing Behavior Analytics

## Project Overview

This project focuses on analyzing retail sales transaction data to understand customer purchasing behavior, identify high-value customer segments, evaluate product performance, and generate actionable business recommendations.

The objective was to transform raw sales data into meaningful insights that can support revenue growth, customer targeting, inventory planning, and operational decision-making.

Through exploratory data analysis, the project investigates how customer demographics, geography, occupation, and product preferences influence purchasing behavior and overall business performance.
